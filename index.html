SELENE

SEa LEvel NEar-real time quality control processing

Nowadays, coastal sea level monitoring and forecasting systems require high-quality tide gauge data in near-real time, with lower samplings and latencies than a few years ago. SELENE software has been developed to minimize the effects of bad data or malfunctions of the sensors on the use of these data and to automatically consolidate flags in a quality controlled time series, minimizing the need of human inspection in near-real time. The software is running operationally at the REDMAR network of Puertos del Estado and, since last January, at the IBI In Situ TAC of the Copernicus Marine Service. Description of the code and use cases are provided here, along with access to the Github created for facilitating control of versions and contributions from other experts/institutions

Quick start

  1. Prepare the environment: Python 3.6.x + Anaconda 3
  2. Install libraries: pandas, scipy, numpy, configparser
  3. Get the gfortran compiler
  4. Download SELENE to your file system
  5. Go to /SELENE/foreman and compile predicc.f with gfortran and options -fno-automatic, -ffixed-line-lenght-132 and -O2
  6. Give execution permissions to predicc.e
  7. Edit /SELENE/configuration/config.ini to indicate where is foreman and outputs folder
  8. Edit /SELENE/configuration/stations.json to add stations and related parameters
  9. Copy sea level time series files to data files folder (these files are configured in stations.json)
  10. Run: python selene.py
  11. View results: python selenevis.py -station
